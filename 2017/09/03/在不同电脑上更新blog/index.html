Set Up Hexo

  1. install nvm
    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.0/install.sh | bash

  2. install node.js
    nvm install v4.4.3

  3. install hexo
    npm install -g hexo-cli

  4. go to the directory where you save your original setting
