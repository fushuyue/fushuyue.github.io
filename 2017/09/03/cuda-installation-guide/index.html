cuda installation guide

Sep 3, 2017 | 深度学习 | Hits
CUDA® is a parallel computing platform and programming model invented by NVIDIA. It enables dramatic increases in computing performance by harnessing the power of the graphics processing unit (GPU).

Prerequsites:

  1. cuda capable GPU
  2. Ubuntu 16.04
  3. gcc

Step One: Install Nvidia Drive

Install build-essential

Pre-install actions

  1. Close secure boot from UEFI BIOS. really important

  2. Enter teletype

  3. Stop lightdm
    sudo /etc/init.d/lightdm stop

  4. Blacklist nouveau drive

Add following lines:

Go back to terminal

Check if nouveau is blacklisted: lsmod | grep nouveau

Download and install drive

Step Two: Install Cuda

  1. Download runfile from the nvidia website
    cuda

  2. Install
    sudo sh cuda_8.0.61_375.26_linux.run

Since we have already install the nvidia drive, we must select not to install drive when installing cuda!

  1. Set up environment variable

edit /etc/profile with the following

Type source /etc/profile in terminal.

In case something went wrong:

  1. Check if cuda is installed: nvcc --version

Step Three: Install Cuda Samples

  1. change to sample directory
    cd /home/xuezhisd/NVIDIA_CUDA-7.0_Samples
  2. make file
    make –j
  3. change to release directory
    cd ./bin/x86_64/linux/release
  4. check if cuda is installed properly
    ./deviceQuery

If installed properly, terminal prints out the information about your GPU.
